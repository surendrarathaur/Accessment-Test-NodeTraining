
RESTful API

An example RESTful API built on Nodejs, Express, Mongoose, JWT

Features:

RESTful routing
Models with proper relationships
JWT Authentication
RESTful errors

Routes List:

Users:

Method                  Action
GET/HEAD	      http://localhost:3000/users/getUsers
POST    	      http://localhost:3000/users/getUser
POST	          http://localhost:3000/users/createUser
PUT	              http://localhost:3000/users/updateUser
DELETE	          http://localhost:3000/users/deleteUser
POST	          http://localhost:3000/users/getUserComments
POST    	      http://localhost:3000/users/getUserPosts

DELETE	          http://localhost:3000/posts/deletePost
